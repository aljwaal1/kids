# طفلي الأول - Kids Learning V1

تطبيق Flutter أندرويد فقط للأطفال من عمر 3 إلى 6 سنوات.

## المحتوى
- العربية: الحروف، الأرقام 1-10، الألوان، الحيوانات.
- English: Capital Letters, Numbers 1-10, Colors, Animals.
- الحيوانات مختلطة: الاسم العربي + الإنجليزي.
- ألعاب بسيطة مع نظام نجوم وإنجازات.
- أصوات عبر محرك النطق في الهاتف TTS، ويمكن لاحقاً استبدالها بملفات صوتية مسجلة.

## البناء عبر GitHub Actions
ارفع المشروع إلى GitHub ثم افتح Actions وسيتم إخراج APK.
