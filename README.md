# طفلي الأول V4

نسخة V4 مبنية فوق النسخة التي كانت تعمل على الهاتف.

## الجديد
- ألعاب تعليمية متعددة بدل سؤال واحد.
- لعبة الحروف العربية.
- لعبة الأرقام العربية.
- لعبة الحروف الإنجليزية.
- لعبة الأرقام الإنجليزية.
- لعبة الحيوانات.
- أسئلة عشوائية وعدم تكرار نفس السؤال مباشرة.
- حفظ النجوم داخل الهاتف بدون أي مكتبات إضافية.
- حفظ أعلى نتيجة وعدد الإجابات الصحيحة.
- صفحة إنجازاتي وجوائز.
- نفس نظام الصوت القديم الذي نجح سابقًا.
