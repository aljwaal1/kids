# طفلي الأول V5

نسخة V5 مبنية فوق نسخة V4 التي كانت تعمل على الهاتف.

## الجديد
- قسم الوالدين لتسجيل الأصوات داخل التطبيق.
- رمز دخول افتراضي لقسم الوالدين: 1234.
- تسجيل صوت خاص للحروف العربية.
- تسجيل صوت خاص للأرقام العربية.
- تسجيل صوت خاص للحروف الإنجليزية.
- تسجيل صوت خاص للأرقام الإنجليزية.
- تسجيل صوت خاص للحيوانات.
- تشغيل التسجيل، إعادة التسجيل، وحذف التسجيل.
- عند الضغط على بطاقة تعليمية يستخدم التطبيق تسجيل الأهل إن وجد، وإلا يستخدم الصوت القديم.
- ألعاب تعليمية متعددة بدل سؤال واحد.
- لعبة الحروف العربية.
- لعبة الأرقام العربية.
- لعبة الحروف الإنجليزية.
- لعبة الأرقام الإنجليزية.
- لعبة الحيوانات.
- أسئلة عشوائية وعدم تكرار نفس السؤال مباشرة.
- حفظ النجوم داخل الهاتف بدون أي مكتبات إضافية.
- حفظ أعلى نتيجة وعدد الإجابات الصحيحة.
- صفحة إنجازاتي وجوائز.
- نفس نظام الصوت القديم الذي نجح سابقًا.
